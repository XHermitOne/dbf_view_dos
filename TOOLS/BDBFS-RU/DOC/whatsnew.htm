Последние изменения

ПОСЛЕДНИЕ ИЗМЕНЕНИЯ 

Обозначения:
	[+] - новая возможность
	[!] - исправлен баг
	[*] - прочие изменени



04.07.2006 1.27f

[+] Переключение на лету кодировок OEM/ANSI (Ctrl+Tab).
    Внимание! Если у кого-то в ini-файлах были определены собственные
    наборы OEM/ANSI (_OEM_SET, _ANSI_SET), их необходимо переопределить
    (ранее использовались частичные наборы - только транслируемые символы,
    сейчас - полные, из 128 символов). Тут же можно задать кодировку баз
    по умолчанию (переменная _Default_Cp). Кодировку конкретной базы можно
    задать в AOP-файле(переменная _DBF_CPage).
    Соответствующие команды -
  	#CODEPAGE, #DEFAULTCODEPAGE.

[+] Параметр командной строки /WIN (/ANSI) определяет кодировку для текущей базы.

[+] По Ctrl+F10 во всех диалогах и редакции полей вызывается таблица символов

[*] В командной строке - можно задать до 5-и параметров.
    Порядок не существенен.

[+] Параметр командной строки /RUN для запуска плагинов.
    Пример: Bdbfs MyBase /RUN=Test().

[+] Колонки можно двигать (менять местами) по клавишам Ctrl+Shift+Left/Right.

[+] Минимизировать, развернуть колонку - Ctrl+H, Ctrl+Shift+H.

[+] При выборе тэга (Alt+T) выражения и условия тэга показываются в
    несколько строчек. Кроме того, в выражениях показывается
    признак уникальности и учитывается реальная	ширина экрана.

[!] Ошибка при выборе тэга, если ранее встретилось многострочное
    вхождение.

[+] В выборе тэгов  - меню по F10.

[!] В плагинах не открывалась по команде USE база не в текущем каталоге.

[!] Ошибка открытия при вызове файла из каталога, содержащего в названии
    точку.

[+] Доступны функции Clipper Tools - CharOne, CharEven, CharOdd.

[+] Функция Out(row, col, cStr, cColor) - главным образом для плагинов.
 
[+] Filer - по F10 показывается полная информация по файлу.

[!] Filer - ошибка при F10, если отсутствовал файл descript.ion.

[!] Filer - в описаниях не показывались длинные имена.

[!] В программе просмотра неправильно показывались файлы, с окончаниями
    строк _LF, а не _CRLF.

[+] Для поиска назад определена горячая клавиша Ctrl+Shift+F7.

[!] При поиске назад и в контекстном неверно показывался индикатор.

[+] Распознаем несколько новых фоксовских кодовых страниц (VFP8).

[+] В контекстном поиске можно задать выражение
    и дополнительное условие (переменная _lGFCond).

[+] Дополнительное меню конфигурации (Система - Установки).

[+] Опции Pack: _lPckNoAsk - паковать без подтверждения
		_lPckCheck - проверять наличие удаленных перед упаковкой.

[+] Возможность выбора из буферабуфера Windows. Управляется
    переменной _ClipWChoice.

[!] Не всегда правильно учитывался параметр _NoAsk (выводить без
    подтверждения) в отчете.

[!] Пункт поиск минимума был по умолчанию закрыт в меню.

[*] В меню преобразования регистра строчки меню показываются
    в соответствующем регистре.

[*] Назначаемые макросы обрабатываются раньше предопределенных.

[!] В редакции MEMO не работала вставка из внутреннего буфера

[!] При редакции MEMO корректно вставляются буферы с переносом строк.

[+] При редакции MEMO Ctrl+K - удаление до конца строки, Ctrl+L - удаление
    слова слева.

[*] При пустом выражении в вычислении "молчим". Ранее выводилось
    предупреждение.

[*] Диапазон действия команд по умолчанию - оставшиеся.

[+] В выборе из буферов (обоих) доступны удаление и передвижка.

[+] В окне BROWSE положение на 1/последней позиции отражается
    спец.символом на вертикальном скроллинге.

[*] При установке фильтра по возможности остаемся на той же записи.






16.10.2003 1.26b

[+] Экспорт в формат MS Excel (XLS). Доступен из меню и
    командной строки по ключу /XLS. См. также информацию об экспорте.

[+] Экспорт в формат XML. Доступен из меню и
    командной строки по ключу /XML. См. также информацию об экспорте.

[+] При экспорте в HTML (а также, XLS и XML) запрашивается
    преобразование Oem->Ansi. Можно отключить запрос путем
    установки конфигурационных переменных.

[+] При организации циклов в инициализационных файлах (плагинах)
    можно использовать операторы LOOP, EXIT.

[!] Могло неправильно отображаться поле типа DateTime при
    установках SET CENTURY OFF или SET DATE ANSI.

[+] При REPLACE поля типа DateTime принимаем формат
    Visual FoxPro ({^yyyy.mm.dd hh:mm}).

[+] При REPLACE поля типа Date принимаем формат FoxPro ({}).

[+] Во всех выражениях распознается формат даты Visual FoxPro
    ({^yyyy.mm.dd}).

[+] К полям типа Currency можно обращаться через $ (например,
    $CurFld). Заносить в них можно через $ ({Alt+R} $111.4).

[*] Ускорена обработка при DELETE/RECALL.

[*] Неполные выражения в полях типа DateTime показываются
    аналогично Visual FoxPro.

[*] Во всех условиях более правильно обрабатываются выражения,
    заданные в инициализационных файлах оператором #xcommand.

[!] Возникала ошибка при занесении в VariField со сменой типа.

[!] Описания (из файла descript.ion) показывались всегда
    прописными буквами.

[+] Можно использовать функции Clipper Tools CharRem(), CharOnly().

[+] Пункт меню "Послать письмо автору".






12.07.2003 1.25b

[+] Поддерживаются поля типа Variant (Clip и Visual FoxPro).

[+] Поддерживаются поля типа Auto Increment (Visual FoxPro).

[+] Поддерживается просмотр контейнера .DBC (Visual FoxPro).

[+] Работа с Clipboard Windows 2000/XP (требует наличия утилиты
    bdb_wclp).

[+] Параметр _NoAsk в печатных формах: не задавать вопросов при
    печати отчета.

[+] Можно использовать функции FoxPro CToBin(), BinToC(), Id(),
    Sys(0), Sys(1), Sys(10), Sys(11).

[!] Иногда в базе убирался признак наличия CDX, что плохо
    отражалось на совместной работе с FoxPro.

[*] Изменен алгоритм сортировки, что привело к увеличению
    скорости на больших базах и уменьшению на маленьких.

[+] Новые макросы в диалогах - Ctrl+F5 - значение текущего поля в
    кавычках, Alt+Shift+" - окавычить набранное.

[+] При выборе файла (базы, индекса и т.д.) на рамочке
    показывается описание из файла descript.ion.

[+] В инициализационных файлах можно использовать операторы
    PUBLIC, PRIVATE, #UNDEF, EXTERNAL, MEMVAR, FIELD.

[+] В операторе #MACRO можно задавать блоки кода.

[!] При чтении инициализационных файлов(плагинов) могла возникать
    ошибка при IF..ELSE, вложенном в WHILE.

[+] Если при чтении структуры обнаружены "неправильные" поля
    или заголовок - предлагаем открыть в форсированном режиме
    (аналогично ключу /forced командной строки).

[*] Более надежно восстанавливаем структуру "форсированного"
    файла при сбоях.

[!] Не вводились отрицательные числа типа VFP Double. Числа типа
    Double без дробной части показывались с точкой справа.

[!] Не вводились и неправильно показывались отрицательные числа
    типа VFP Currency.

[!] В показе структуры (Alt+F2) неправильно показывалась длина
    полей типа Currency, Double.

[+] Вызов сайта поддержки из меню. В системе должен быть
    определен браузер по умолчанию.

[*] Более информативное сообщение при запуске без параметров или
    с параметром /?.





Описание более ранних изменений будут высланы желающим.
